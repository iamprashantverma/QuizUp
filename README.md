## How to Run Locally
1. **Clone the repository:**
```bash
git clone https://github.com/iamprashantverma/QuizUp.git
```
2. Navigate to the project directory:
```bash
cd QuizUp
```
3. Open index.html in your browser.

4. Click "Start Quiz" and begin answering questions
